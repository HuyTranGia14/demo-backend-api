# Demo Backend API

Practice project để học routing và controllers với FastAPI.

## Chạy project:

```bash
# Activate venv
.\Scripts\Activate.ps1

# Run server
python demo_api.py
```

## Endpoints:

- `GET /` - Hello world
- `GET /login?username=xxx&password=xxx` - Login
- `GET /profile/{username}` - User profile
